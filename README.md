# Quiz master
Train your brain and challange your knowlage with this quiz game made in Unity.

## 🛠️ Technical details
* **Question selection** -> all the questions are selected randomly with the help of some game objects that contain all the necessary details abot the certain questions
* **Time System** -> the time is represented by the circle in the UI (30 seconds to answer, 10 seconds waiting time)
* **Answer a question** -> when you press a button representing an answer, your answer is compered with the correct anwer noted in the game object of the question (yourAnswer == correctAnswer (you answerd correctly), yourAnswer != correctAnswer (wrong answer))
* **Score System** -> correctAnswers / questionsUntilNow * 100
* **End screen** -> after you completed all the questions, you can try again the quiz, but the questions will be randomized again
* !! I don't own the assets.
